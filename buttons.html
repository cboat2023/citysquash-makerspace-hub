Key vocabulary

Input
Information that enters the Arduino, such as a button press.
Output
An action the Arduino controls, such as turning on an LED.
Pushbutton
A switch that changes the circuit while it is pressed.
Pull-down resistor
Keeps the input at a steady LOW signal when the button is not pressed.
digitalRead()
Checks whether a digital input signal is HIGH or LOW.
