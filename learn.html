Debugging checklist

Before Asking for Help

  1. Is the circuit powered?
  2. Is there a complete path from power to ground?
  3. Are the wires in the correct breadboard rows?
  4. Is the LED facing the correct direction?
  5. Is a resistor included?
  6. Does the code use the correct pin number?
  7. Did you start the simulation?
  8. Are there any error messages?
  9. What is one part you can test separately?
  10. What did you already try?

Team-first rule: Ask another student for help before asking the instructor, unless the issue involves safety, heat, smoke, damaged equipment, or a power problem.
