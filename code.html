Key vocabulary

setup()
Runs once when the Arduino starts. Use it to prepare pins and settings.
loop()
Repeats again and again while the Arduino has power.
pinMode()
Tells the Arduino whether a pin will be used as an input or output.
digitalWrite()
Sends an on or off signal to a digital output pin.
delay()
Pauses the program for a number of milliseconds.
HIGH
Turns a digital output signal on.
LOW
Turns a digital output signal off.
Hands-on challenge

Change the Blink Pattern

  1. Change both delay() values.
  2. Run the simulation.
  3. Describe what changed.
  4. Try one short delay and one long delay.
  5. Create your own recognizable blink pattern.
